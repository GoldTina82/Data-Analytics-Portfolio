# 📊 Treasury Cash Flow Forecast & Dashboard (Excel)

## 🔹 Overview

This project is an Excel-based **Treasury Cash Flow Forecast model** designed to track daily cash inflows and outflows, monitor liquidity, and identify potential **cash gaps**.

It simulates a real-world financial scenario and demonstrates how Excel can be used for **financial analysis and decision-making**.

---

## 🔹 Project Files

* `treasury-cashflow-dashboard.xlsx` → Main Excel model with forecast, analysis, and dashboard

---

## 🔹 Key Components

### 1. Cash Flow Forecast

* Daily tracking of **Cash In** and **Cash Out**
* Calculation of:

  * Opening Balance
  * Net Cash Flow
  * Closing Balance

---

### 2. Liquidity Analysis

* Classification of:

  * OK
  * Low Liquidity
  * Cash Gap

---

### 3. Dashboard

* KPIs:

  * Total Cash In
  * Total Cash Out
  * Net Cash Flow
  * Final Balance
* Charts:

  * Cash Flow Trend
  * Category Analysis

---

### 4. Data Analysis

* Pivot Tables for:

  * Cash flow by category
  * Expense breakdown

---

## 🔹 Excel Techniques Used

* Structured References (Excel Tables)
* SUM / SUMIFS / COUNTIF
* Conditional Formatting
* Pivot Tables & Charts
* **Modern Excel functions (e.g. TAKE for dynamic last value retrieval)**

---

## 🔹 Key Insight

This model highlights how **timing differences between inflows and outflows** can create liquidity issues, even when total inflows are higher than outflows.

---

## 🔹 Business Value

* Helps monitor **cash position in real time**
* Supports **financial planning**
* Identifies **liquidity risks early**
